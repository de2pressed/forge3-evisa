# Forge eVisa

A cinematic, human-first redesign of India’s normal e‑Business visa journey.

## Pages

- `index.html` — landing and route finder
- `apply.html` — five-stage application
- `eligibility.html` — eligibility and requirements
- `help.html` — support and FAQs
- `track.html` — status and document recovery
- `eta.html` — approved digital ETA

Open `index.html` directly or serve the folder with any static web server. The site has no framework or runtime dependencies.

Project context starts in `AGENTS.md`; the complete hackathon build record is in `process.md`.
